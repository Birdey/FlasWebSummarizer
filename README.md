# WebViewer

WebViewer is a Python-based web application for viewing and analyzing web content.

## Features

- View web pages
- Analyze web content
- User-friendly interface

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/birdey/FlasWebSummarizer.git
    ```
2. Navigate to the project directory:
    ```sh
    cd webviewer
    ```
3. Install the required dependencies:
    ```sh
    pip install -r requirements.txt
    ```

## Usage

1. Run the application:
    1. On Windows:
        - Double-click 'run_flask_app.bat'
        - Open a command prompt and run:
        ```sh
        set FLASK_APP=webviewer
        set FLASK_ENV=development
        flask run
        ```
    2. On Linux or macOS:
        ```sh
        export FLASK_APP=webviewer
        export FLASK_ENV=development
        flask run
        ```

2. Open your web browser and go to `http://localhost:5000`.
